 # Zeldra Project
 
 This project is maven archetype for helping generate spring boot template in BE side.  
 
### Command To Use

```
mvn archetype:generate \
    -DarchetypeGroupId=com.brightkut \
    -DarchetypeArtifactId=zeldra \
    -DarchetypeVersion=1.0-SNAPSHOT 
```
